# 数量关系

## 奇偶特性

两个数的和若为奇数，那么两个数的差也为奇数

同理可得

两个数的和若为偶数，那么两个数的差也为偶数

## 整除特性

一个数能被2（5）整除，末一位能被2（5）整除

一个数能被4（25）整除，末两位能被4（25）整除

一个数能被8（125）整除，末三位能被8（125）整除

一个数能被3（9）整除，其各位数字之和能被3（9）整除
